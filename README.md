# REST API Development with Raw JSON Data

## Overview

This project demonstrates the development and implementation of REST APIs using raw JSON text data. It focuses on performing basic CRUD (Create, Read, Update, Delete) operations while following RESTful principles and exchanging data in JSON format.

## Features

* RESTful API endpoints
* CRUD operations on JSON data
* JSON request and response handling
* Input validation and error handling
* Lightweight implementation for learning and practice

## Technologies Used

* Java
* Spring Boot
* REST API
* JSON

## API Operations

* **GET** – Retrieve data
* **POST** – Create new records
* **PUT** – Update existing records
* **DELETE** – Remove records

## Purpose

The primary goal of this project is to gain hands-on experience in designing, implementing, and testing REST APIs while working directly with raw JSON data and understanding the request-response lifecycle.
